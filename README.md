# FreeAPI Authentication App

A simple authentication-based application built with HTML, Tailwind CSS, and vanilla JavaScript using the FreeAPI Authentication Module.

## Features

- **User Registration**: Create new user accounts
- **User Login**: Sign in with existing credentials
- **Profile Display**: Show details of the currently logged-in user
- **User Logout**: End the user session
- **Loading States**: Visual feedback during API requests
- **Success/Error Messages**: Informative notifications for user actions
- **Clean UI**: Minimal black and white design using Tailwind CSS

## Technologies Used

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- FreeAPI (https://api.freeapi.app)

## API Endpoints Used

- **Register User: POST https://api.freeapi.app/api/v1/users/register
- **Login User: POST https://api.freeapi.app/api/v1/users/login
- **Logout User: POST https://api.freeapi.app/api/v1/users/logout
- **Get Current User: GET https://api.freeapi.app/api/v1/users/current-user

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Register a new account or login with existing credentials

## Project Structure

- `index.html`: Main HTML file with UI structure
- `app.js`: JavaScript file with authentication logic
